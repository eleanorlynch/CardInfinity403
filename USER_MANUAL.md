This guide will help you get started and set up Card Infinity on Discord. 

Card Infinity is a free online customizable Discord Activity that allows you to play card games with your friends. What makes it unique is that you can customize rules for an existing game, or you can even create your own card game if you desire. The interface shows you the main "table" with the draw and discard piles, as well as your personal hand of cards. It's designed to be an easy and fun way to play together with your friends on Discord. 

How to Install 

Card Infinity runs in Discord, so make sure you have Discord installed and set up on your device. Discord can be found in the App Store or Google Store for mobile devices, and here is a guide to installing Discord on desktop, Desktop-Installation-Guide 

Make sure you make a Discord account and also enable 2FA. Here is a guide for that, 2FA (Two-Factor Authentication)
You can then enter a chat and launch Card Infinity from the Activities center. 

Quick Start (using our link) 
Click on the link to install Card Infinity Card Infinity
Select the server where you want to add Card Infinity, or choose "Add to Apps" to use it anywhere.
To launch the game on a server, go to any text or voice channel, click the Activities button in the chat bar, and select Card Infinity from the list. To launch on a chat or group chat, open the chat, click the Activities button, and select Card Infinity from the list.
Once a host starts the game (aka you) other people in the channel can join your game by clicking “join,” and now you can all play together 

Use the Software 
When you first launch Card Infinity, you will see the main screen with three options 
Start Game- Creates a new game lobby 
You are now the Host, and you can select the ruleset to use. You can pick an existing one, or you can pick a custom ruleset you’ve created (this is a work in progress) 
Manage Rules- Opens the ruleset editor, where you can create or modify custom rulesets (this is a work in progress) 
Join Game - Allows you to enter an existing game that someone else is hosting 
Playing the Game 
Currently, the default game is a simplified version of Uno, you will see a table with cards and your hand of cards. If you can, you play a card, if you can't, you click “draw” to draw a card and then click “end turn”. You must click “end turn” after any action to pass the turn to the next player.

Frequently Asked Questions
Q: Do I need to install anything other than Discord?
A: Nope! Card Infinity runs entirely inside Discord. Just click the installation link, and you're good to go.
Q: Can I play on mobile?
A: Discord Activities work best on desktop. Mobile support is limited but may work in some cases. 
Q: How do I create my own rules?
A: Click "Manage Rules" from the main menu. The UI for creating rules is currently a work in progress, but you can see what options will be available soon.
